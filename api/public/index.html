⏱ Tick Control▶
Weather affects ALL blocks every tick:
• Base damage: 5 HP per tick (always, regardless of weather)
• Rain damage: floor(rain_mm × 10) extra HP per tick
• Wind damage: floor(wind_speed_kph ÷ 3) extra HP on windward-edge blocks only
  → Wind direction determines which grid edge takes the bonus hit
  → e.g. Wind=N damages blocks in row y=0; Wind=E damages blocks in column x=19

Block HP: dry_sand=25, wet_sand=40, packed_sand=60. Reinforce adds 15 HP (max 60).
Rain (mm) — adds floor(rain × 10) damage to every block
Wind speed (kph) — adds floor(speed ÷ 3) damage to windward-edge blocks
Wind direction — determines which grid edge takes wind damage
